NAME

fwupdate -ddebugging utility for UEFI firmware updates

SYNOPSIS

fwupdate [CMD]

DESCRIPTION

fwupdate is a deprecated tool that allows deploying capsule updates.

OPTIONS

The fwupdate command takes various options depending on the action. Run fwupdate --help for the full list.

fwupdtool(1), fwupdmgr(1)
