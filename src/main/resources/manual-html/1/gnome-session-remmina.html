NAME

gnome-session-remmina — GNOME Shell session script for Remmina

DESCRIPTION

gnome-session-remmina This script is part of the Remmina kiosk mode. If you have a freedesktop compliant Login Manager, a Remmina kiosk mode will be added in the list of available sessions.
