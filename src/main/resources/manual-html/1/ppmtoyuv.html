NAME

ppmtoyuv - convert a portable pixmap into an Abekas YUV file

SYNOPSIS

ppmtoyuv [ppmfile]

DESCRIPTION

Reads a portable pixmap as input. Produces an Abekas YUV file as output.

SEE ALSO

yuvtoppm(1), ppm(5)
