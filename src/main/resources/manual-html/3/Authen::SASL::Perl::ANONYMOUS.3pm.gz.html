NAME

Authen::SASL::Perl::ANONYMOUS - Anonymous Authentication class

SYNOPSIS

use Authen::SASL qw(Perl);
$sasl = Authen::SASL->new(
mechanism => 'ANONYMOUS',
callback => {
authname => $mailaddress
},
);

DESCRIPTION

This method implements the client part of the ANONYMOUS SASL algorithm, as described in RFC 2245 resp. in IETF Draft draft-ietf-sasl-anon-XX.txt.

CALLBACK
The callbacks used are:
authname

email address or UTF-8 encoded string to be used as trace information for the server
