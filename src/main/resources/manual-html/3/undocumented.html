SYNOPSIS

Undocumented library functions

DESCRIPTION

Solicitation
If you have information about these functions, please look in the source code, write a man page (using a style similar to that of the other Linux section 3 man pages), and send it to mtk.manpages@gmail.com for inclusion in the next man page release.
