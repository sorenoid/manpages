NAME

rpc_gss_set_defaults — set service and quality of protection

SYNOPSIS

#include <rpc/rpcsec_gss.h>

bool_t

rpc_gss_set_defaults(AUTH *auth, rpc_gss_service_t service, const char *qop);

DESCRIPTION

Set the service and quality of protection to be used for RPC requests. The new values apply for the rest of the lifetime of the context (unless changed again with this function).

PARAMETERS
service

The service type to use for subsequent RPC requests

qop

The quality of protection to use or NULL for the default

RETURN VALUES

Returns TRUE if the values were set

AVAILABILITY

The rpc_gss_set_defaults() function is part of libtirpc.

SEE ALSO

rpc(3), gssapi(3), rpcsec_gss(3)
